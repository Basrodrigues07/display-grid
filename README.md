# Organic Store Website

## Sobre o Projeto

Este projeto foi desenvolvido como parte de uma atividade escolar proposta pelo professor, onde os alunos deveriam recriar uma página web utilizando HTML e CSS. O desafio consistia em reproduzir um layout de e-commerce de produtos orgânicos com design responsivo e funcionalidades modernas.

## Objetivo da Atividade

- Aplicar conhecimentos de HTML5 e CSS3
- Desenvolver habilidades de layout responsivo
- Implementar boas práticas de desenvolvimento web
- Criar uma experiência de usuário intuitiva e moderna

## Funcionalidades Implementadas

### Páginas Desenvolvidas
- **Home** - Página principal com produtos em destaque
- **Shop** - Catálogo de produtos com filtros
- **Blog** - Artigos sobre vida orgânica e receitas
- **Contact** - Formulário de contato e informações da loja

### Recursos Técnicos
- Design responsivo para desktop e mobile
- Grid CSS para layouts flexíveis
- Formulários funcionais com validação HTML5
- Navegação intuitiva entre páginas
- Hover effects e transições suaves
- Integração com Google Maps
- Deploy automático via GitHub Pages

## Tecnologias Utilizadas

- **HTML5** - Estrutura semântica das páginas
- **CSS3** - Estilização e layout responsivo
- **Google Fonts** - Tipografia (Cairo)
- **GitHub Pages** - Hospedagem e deploy

## Estrutura do Projeto

```
display-grid/
├── index.html          # Página principal
├── shop.html           # Página da loja
├── blog.html           # Página do blog
├── contact.html        # Página de contato
├── style.css           # Estilos da página principal
├── shop.css            # Estilos da página da loja
├── blog.css            # Estilos do blog
├── contact.css         # Estilos da página de contato
└── assets/             # Imagens e recursos
```

## Design e Layout

O projeto segue um design moderno e limpo com:
- **Paleta de cores** baseada em verde (#7fad39) para transmitir naturalidade
- **Tipografia** clara e legível com a fonte Cairo
- **Layout responsivo** que se adapta a diferentes tamanhos de tela
- **Componentes reutilizáveis** para manter consistência visual

## Responsividade

O site foi desenvolvido com abordagem mobile-first, garantindo:
- Navegação adaptável para dispositivos móveis
- Grids flexíveis que se reorganizam conforme o tamanho da tela
- Imagens otimizadas para diferentes resoluções
- Formulários e botões adequados para touch
